## Использование
Простейший вариант запуска через docker:

Введите команду для билда образа:
```sh
docker build -t innopolis-final .
```

После запустите контейнер:
```sh
docker run -p 8000:8000 innopolis-final
```
### Требования
Для установки и запуска проекта, необходим ffmpeg: https://www.ffmpeg.org/download.html#build-windows
